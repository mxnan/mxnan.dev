## Created a Custom-scrollbar

you can customize this further

## Example


> globals.css

```css
::-webkit-scrollbar-thumb {
  @apply bg-secondary;
}
::-webkit-scrollbar-track {
  @apply bg-tertiary;
}
::-webkit-scrollbar {
  @apply w-2 h-1 rounded-xl;
}
::-webkit-scrollbar-button {
  @apply bg-primary;
}
```
